#### BUNDLE-001: Flutter Desktop Integration
**Goal:** Single `nexus-open` binary that includes the UI

**Approach 1: Embed Flutter Bundle (Recommended)**
```go
//go:embed ui/build/linux/x64/release/bundle

func launchFlutter() {
    // Extract flutter bundle to temp dir
    // Launch flutter app as subprocess
}
```

**Approach 2: Separate Processes (Simpler for now)**
```go
// Backend launches Flutter UI automatically
func (a *App) startUI() error {
    cmd := exec.Command("flutter", "run", "-d", "linux")
    cmd.Dir = "ui"
    return cmd.Start()
}
```

## 📐 Technical Architecture Changes

### Display Rendering Stack
```
Before:
┌──────────────┐
│ renderer.go  │ → basicfont → ugly text
└──────────────┘

After:
┌──────────────────────────────────────┐
│ renderer.go                          │
│  ├─ TrueType font loading           │
│  ├─ Embedded icon sprites (go:embed)│
│  ├─ Layout system (sections)        │
│  └─ Animation system (smooth trans) │
└──────────────────────────────────────┘
```

### Application Launch Flow
```
Before:
Terminal → go run . (backend)
Terminal → cd ui && flutter run (frontend)

After:
Menu → nexus-open → ┌─ backend (Go)
                     ├─ API server
                     └─ UI (Flutter subprocess)
                     └─ System tray
```

```
Before:
┌────────────┐

┌─────────────────────────────┐

│ │  ├─ Device preview        │
│ │  └─ Quick actions         │
```

## ⚠️ Known Challenges

### Challenge 1: Font Rendering at Small Resolution
**Problem:** 640x48 is tiny, fonts may not look good
**Solution:**
- Test multiple fonts and sizes
- Consider custom bitmap fonts if TrueType doesn't work
- May need anti-aliasing adjustments

### Challenge 2: Flutter Embedding
**Problem:** Embedding Flutter in Go binary is complex
**Solution:**
- Use subprocess approach for v0.1.5
- Defer full embedding to v1.0
- Works well enough for now

### Challenge 3: System Tray on Wayland
**Problem:** System tray doesn't work well on Wayland
**Solution:**
- Test on both X11 and Wayland
- Provide fallback (keep window open)
- Document limitations
